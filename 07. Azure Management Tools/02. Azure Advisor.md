# Azure Advisor

Azure Advisor is a free service built into Azure that provides recommendations on
- high availability
- security
- performance
- cost

It analyzes your deployments and provides information on how you can improve them across those 4 mentioned areas.
With Azure advisor you can,
- Get proactive, actionable, and personalized best practices recommendations
- improve high availability, security and performance
- identify opportunities to reduce costs
- get recommendations with proposed actions in-line.
